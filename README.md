# Weather App 

**Descrição / Description**  
Aplicativo de clima desenvolvido com HTML, CSS e JavaScript. Permite pesquisar qualquer cidade e ver o clima atual, incluindo temperatura, condição do tempo e um ícone representando o clima.  
Weather App built with HTML, CSS, and JavaScript. Allows you to search any city and see the current weather, including temperature, weather condition, and an icon.

---

## Funcionalidades / Features
- Pesquisar clima de qualquer cidade / Search weather for any city  
- Mostrar temperatura, condição do clima e ícone / Display temperature, weather condition, and icon  
- Design responsivo / Responsive design  

---

## Tecnologias / Technologies
- HTML  
- CSS  
- JavaScript  
- API de clima (ex: OpenWeatherMap) / Weather API (e.g., OpenWeatherMap)  

---

## Como usar / How to Use
1. Clone ou baixe este repositório / Clone or download this repository  
2. Abra `index.html` no navegador / Open `index.html` in your browser  
3. Digite uma cidade na caixa de busca e pressione Enter / Type a city in the search box and press Enter
